## Quem sou

Sou Engenheiro de Computação com atuação **Full Stack** — construo soluções completas, do design de interfaces ao deploy em produção. Trabalho com **Java (Spring Boot), React, Node.js e TypeScript** no dia a dia, e tenho experiência crescente em **Cloud AWS e automação de infraestrutura (DevOps/IaC)**.

Sou mestrando em Engenharia Elétrica e formado também em Matemática. Esse background me dá uma forma de pensar estruturada: decomponho problemas complexos em partes claras e entrego soluções que fazem sentido do ponto de vista técnico e prático.

Nos últimos anos atuei como **freelancer Full Stack** — o que significa que aprendi a entender o negócio do cliente, transformar isso em requisitos técnicos, construir a solução e colocar no ar. Trabalhei com ERPs, sistemas de gestão, cardápios digitais, apps mobile e backends serverless.

Fora do código, sou professor de matemática há 5 anos — o que me tornou uma pessoa que se comunica bem, explica com clareza e tem muita paciência para encontrar a solução certa.

### [📱 Matemática+](https://github.com/luisFernandoJv/matematica-mais) — App Mobile Educacional com Backend Serverless
> React Native (Expo) · TypeScript · AWS Lambda · API Gateway · DynamoDB · Cognito · GitHub Actions

### [🏭 Império Sucata ERP](https://github.com/luisFernandoJv/Imperio-Sucata-ERP) — Sistema de Gestão de Estoque

### [🔌 API REST com Padrão MVC e Repository](https://github.com/luisFernandoJv/nodejs-mvc-repository-pattern) — Node.js + MySQL

API construída com foco em arquitetura: MVC + Repository Pattern para desacoplar camadas, facilitar testes e garantir escalabilidade. Mostra como penso sobre organização de código em projetos backend.

### [🐧 Infraestrutura como Código — Servidor Web com Shell Script](https://github.com/luisFernandoJv/linux-project-2-iac-)
